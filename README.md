# 📺 Canales IPTV & Películas M3U

Repositorio con listas M3U para reproducir canales de TV en vivo y películas en cualquier reproductor compatible (VLC, IPTV Smarters, TiviMate, etc).

## 📁 Archivos

| Archivo | Contenido |
|---------|-----------|
| `DemoApp.m3u` | Canales deportivos en vivo (ESPN, Fox Sports, TNT Sports, TyC Sports, Win Sports, etc.) |
| `PeliculasEstrenos.m3u` | Películas de estreno (2025) |

## 🚀 Cómo usar

1. Copia el enlace raw del archivo `.m3u` que quieras usar
2. Abrilo en tu reproductor favorito:
   - **VLC**: Medio → Abrir ubicación de red → Pegar URL
   - **IPTV Smarters**: Agregar playlist → URL del archivo
   - **TiviMate**: Agregar playlist → M3U playlist → Pegar URL

### URLs directas (raw)

```
https://raw.githubusercontent.com/virtualgaby361-blip/CanalesIPTV/main/DemoApp.m3u
```
